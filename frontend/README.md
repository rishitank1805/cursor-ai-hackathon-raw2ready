# Raw2Ready Frontend

A React application for transforming raw business ideas into perfect pitches with presentations and demos.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
